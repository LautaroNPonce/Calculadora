# JS Calculator

Calculadora web desarrollada con HTML, CSS y JavaScript.

## 📌 Funcionalidades
- Suma
- Resta
- Multiplicación
- División
- Validación de división por cero

## 🛠 Tecnologías
- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo
Aplicar lógica de programación en JavaScript para realizar operaciones matemáticas básicas en una interfaz web.
